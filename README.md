# AI-Based Dynamic Skincare Recommender System

## Brief Description

The AI-Based Dynamic Skincare Recommender System is a smart web application designed to provide personalized skincare recommendations based on user skin type, concerns, and preferences. The project helps users discover suitable skincare routines and products using an interactive and user-friendly interface.

This project aims to simplify skincare selection by using dynamic recommendation logic and modern web technologies.

# Installation / Execution Steps

## Step 1: Clone the Repository

```bash
git clone https://github.com/chhavi0625/ai-skincare-recommender.git
```

---

## Step 2: Open the Project

Open the project folder in VS Code.

---

## Step 3: Install Dependencies

```bash
npm install
```

---

## Step 4: Run the Project

```bash
npm run dev
```

---

## Step 5: Open in Browser

Open the local server link shown in the terminal.

Example:

```text
http://localhost:5173
```
